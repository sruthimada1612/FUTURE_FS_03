# Task 3: Local Business Website & Live Pitch Project

This repository houses the consumer-facing production landing page designed and optimized for "The Urban Bistro" restaurant to drive physical table bookings and catalog engagement.

## 🛠️ Tech Stack
- **Frontend Framework:** React.js, High-Fidelity CSS Grid Systems

## 🚀 Key Features
- **Categorized Digital Menu:** A highly responsive grid layout displaying items by category (Breakfast, Main Course, Desserts, Beverages) with flat latency under 200ms.
- **Dynamic Booking Engine:** A stateful table reservation intake funnel supporting layout variable selections (Guest size selection, reservation verification states).
- **Conversion-Optimized UI:** Designed to replace slow, static PDF menus with actionable interactive local business search capture fields.
